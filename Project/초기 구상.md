# 최종 프로젝트 

###### 3조 프로젝트 만들러 가재영~ 겨레가 다할겨레~



## 1. 개요

영화를 주제로한 서비스 구축



## 2. 기본 틀(배운내용)

### (1) 회원

#### A. 로그인

modal 창으로 로그인창 뜨게하고 회원가입 버튼 누를 시 signup 페이지로 이동

#### B. 회원가입

구현

#### C. 내 정보 페이지

내가 작성한 글, 댓글 모아보기 기능

내가 좋아요한 영화 목록 확인

좋아요, 팔로우 확인

---

### (2) 메인

그리드시스템 적용

#### A. 홈화면

최신 개봉한 영화들을 carousel로 넘겨가며 보여주기

#### B. 영화 목록

영화들을 보여주고 클릭 시 상세페이지 이동

---

### (3) 상세 페이지

#### A. 상단 영화 정보

영화에 대한 디테일한 정보와 예고편 보여주기

영화 좋아요 구현(비동기식)

#### B. 하단 영화 리뷰

하단에 리뷰 CRUD-댓글 구현

리뷰 좋아요, 유저 팔로우 구현(비동기식)

---



## 3. 추가적인 아이디어

회원 기능에 활동 로그 확인 시스템 구현 - 좋아요 누른 것, 검색한 것, 팔로우 누른것, 취소한 것 등등

영화목록에서 인기순, 평점순, 개봉일자순 등등의 정렬 시스템 구현
