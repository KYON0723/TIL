## 2. 기본 틀(배운내용)

### (1) 회원

#### A. 로그인

modal 창으로 로그인창 뜨게하고 회원가입 버튼 누를 시 signup 페이지로 이동

성공시 새로고침

#### B. 회원가입

구현

#### C. 내 정보 페이지

내가 작성한 글, 댓글 모아보기 기능

내가 좋아요한 영화 목록 확인

### (2) 메인

그리드시스템 적용

#### A. 홈화면

#### B. 영화 목록

영화들을 보여주고 클릭 시 상세페이지 이동

## 3. 추가적인 아이디어

회원 기능에 활동 로그 확인 시스템 구현 - 좋아요 누른 것, 검색한 것, 팔로우 누른것, 취소한 것 등등

영화목록에서 인기순, 평점순, 개봉일자순 등등의 정렬 시스템 구현

정렬시스템에 추가적으로 상세필터 적용으로 평점 00 이상, 이하 개봉일 00 이전 이후 등등도 구현

스포일러 포함된 리뷰나 글일 경우 글 작성시 옵션으로 넣고 다른사람이 글을 보려할때 경고창이 뜨도록

글을 확인했는데 스포일러라면 이용자들이 투표를 하여 일정이상 누적시 자동으로 스포일러방지 처리되도록

게시판 공지사항을 맨 위에 고정으로 보이게하고 이용규칙 작성

신고시스템을 도입하여 일정이상 받을 시 내 정보에서 건전한 활동을 해주세요 경고문구 출력

글쓰고 댓글쓰고 하면 경험치 쌓여서 레벨업하는 시스템
